*Buildar e Rodar
docker-compose up --build

*Parar
docker-compose down

*TypeORM como ORM para Postgres.
*Zod para criação de validação campos.
*Middleware ErorHandler com criação de classes de erro para resposta padronizadas da API e captura de erros inesperados automaticamente.
*Clean Architecture utilizando padronização de desenvolvimento por useCases.
*Swagger para documentação da API.
